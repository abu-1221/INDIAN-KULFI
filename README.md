An enterprise-grade administrative and operational portal designed for high-precision management of inventory, sales, and fiscal records for a multi-product frozen dessert business.

---

## 🚀 Key Features

### 💎 Premium Responsive UI
*   **Mobile-First Design**: Fully optimized for phones, tablets, and desktops.
*   **Glassmorphism Theme**: A sleek, dark-mode aesthetic using Tailwind CSS for a professional feel.
*   **Zero-Cutoff Layout**: Intelligent scrolling ensuring all data is visible on any screen size.

### 📊 Operational Intelligence
*   **Sales Terminal (POS)**: Rapid transaction dispatch with real-time stock synchronization.
*   **Warehouse Matrix**: Real-time inventory tracking with critical "Low Stock" alerts.
*   **General Ledger**: Consolidated chronological financial record with a live Running Balance.
*   **Live Dashboard**: Real-time charts and KPI monitoring.

### ⚙️ Seamless Automation
*   **One-Click Startup**: Silent background bootloader (`Launch_Kulfi.vbs`) that clears ports automatically.
*   **Cloud Registry**: Production-grade hosting on Render.com for global access and centralized data.
*   **Shop Cloud (Local Hosting)**: Bind to your local Wi-Fi for **Zero Cost** and **Zero Lag** performance across all shop devices.
*   **Auto-IP Detection**: Startup script automatically displays the network URL for mobile connections.

---

## 🛠️ Tech Stack
*   **Backend**: Django (Python)
*   **Frontend**: Vanilla JS, HTML5, CSS3 (Tailwind CSS)
*   **Database**: SQLite (ACID Compliant)
*   **Automation**: VBScript, Batch Scripting
*   **API**: REST Framework for mobile synchronization

---

## 🚦 Quick Start

1.  **Installation**: Run `setup.bat` to initialize the environment and install dependencies.
2.  **Launch**: Double-click `Launch_Kulfi.vbs` to start the server in the background.
3.  **Access**:
    *   **Local PC**: Open `http://localhost:8000`
    *   **Mobile**: Use the IP address shown in the startup popup (e.g., `http://192.168.1.5:8000`).
4.  **Termination**: Run `stop.bat` to shut down the background engine.

---

## 📱 Mobile Connection Guide
1.  Ensure your phone and PC are on the **same Wi-Fi network**.
2.  Open your mobile browser.
3.  Enter the URL displayed in the startup notification.
4.  Experience high-speed, zero-lag management from your pocket!
